# ML Workflow Notes

This notebook documents the exploratory machine learning workflow used in the translational immunology project.

- classify cytokine stability patterns
- identify matrix-specific predictive features
- explore cytokine behavior across freeze-thaw conditions

## ML Methods

## Features Used

The project investigates how repeated freeze-thaw cycles influence multiplex cytokine assay stability across EDTA plasma, heparin plasma, and serum matrices.
